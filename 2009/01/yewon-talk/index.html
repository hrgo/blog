딸기 마주이야기

딸기랑 목욕하며 한 대화.

이모 : 딸기야 한국에 있을때 할머니 산소 간거 생각나?
딸기 : 응
이모 : 거기 가서 뭐했어?
딸기 : 거기 뽈록 나온데 올라갔어 (산소를 말하는듯)
이모 : 거기 뽈록나온데 그게 뭔데?
딸기 : (우물쭈물)
이모 : 거기 뭐 들어 있어?
딸기 : 할머니
이모 : 어? 할머니는 하늘나라에 있다며~ 이상하네?? 그럼 거기 뽈록나온데는 뭐가 있는거지??
딸기 : 할머니 몸
이모 : 그럼 하늘나라엔 뭐가 있어? (영혼이란 말을 기대하며)
딸기 : 얼굴!

목욕탕이 쩌렁쩌렁 울리도록 한참 웃었다. 당분간 외롭거나 슬플땐 이걸 생각하련다.
