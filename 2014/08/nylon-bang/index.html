게임의 목표

본인에게 유리한 점수에 판을 끝내거나 다른 사람이 판을 끝냈을 때 낮은 점수를 사수한다. 따라서 좋은패란 낮은 점수 또는 마이너스 점수를 낼 수 있는 패. (그래서 주로 초반은 비 똥 등의 높은 숫자가 버려짐.)

게임 진행 방법

  • 화투 또는 카드로 가능 (12까지의 숫자가 식별 가능해야 함)
  • 일정 판 수를 정하고 시작. (보통 10판)
  • 이전 판의 꼴찌가 선을 잡음.
  • 5장씩 나눠주고 선은 6장을 갖고 시작.
  • 선부터 고도리 방향으로 돔.
  • 한장 가져가고 필요한 것을 갖고 필요없는 것 한장을 내려놓음.
  • 게임이 끝날 때까지 계속 돌다가 누군가 스탑 또는 바가지를 할 경우 게임이 끝남.
  • 각 판이 끝날 때 각자 손에 든 화투장의 점수를 합산해서 가장 점수가 낮은 사람부터 순위가 매겨짐.

용어 설명

  • 뻥 : 누가 화투장을 버릴 때 본인이 두장 들고 있으면 뻥을 씌울 수 있음. 그리고 나서 한장 더 버리므로 두 장의 화투장으로 플레이 하게 됨. 뻥을 하고 나면 두 장의 패만 맞추면 되므로 바가지나 스톱으로 게임을 끝낼 가능성이 커짐. eg. A선수가 11을 버릴 때 순서에 상관없이 누구든 11을 두장 들고 있으면 “11뻥” 을 씌우고 가진 화투장 중 필요 없는 것을 한 장 버림. 게임은 뻥을 씌운 사람 다음 사람부터 다시 돌아감.
  • 바가지 : 뻥을 하고 손에 두장을 들은 상태에서 하는 뻥은 바가지라 함. eg. A선수가 3을 버렸는데 B선수가 3을 두장 들고 있다가 뻥을 씌우면서 화투장을 털어버리면 A는 바가지를 씀. 
  • 자연뻥 : 같은 숫자를 3장 가지면 자연뻥 즉, 0로 계산되고 석장을 뺀 나머지만 합산. (다섯장을 들고 그중 세장이 자연뻥이라면 다섯장을 들고도 바가지를 씌울 수 있음. ) eg. A선수가 5,5,5,2,4를 들고 있다가 누군가 스탑하면, 5,5,5는 자연뻥으로 0점. 나머지 2,4만 합산. 
  • 스트레이트 :  6장으로 숫자가 나란히 맞춰진 경우. (훌라와 달리 12에서 1로는 연결되지 않음.) eg. A선수가 2,3,4,5,6 를 들고 있다가 자기 차례에 한장을 가져갔는데 1 또는 7이 나온 경우. 또이또이 : 6장으로 둘둘둘, 셋셋 또는 넷둘 로 짝을 맞춘 경우.  eg. A선수가 3,3,7,7,9,9 를 들고 있다가 자기 차례에 한장을 가져갔는데 9가 나온 경우 둘둘. 4,4,4,5,5,5 셋셋, 6,6,6,6,5,5 넷둘.
  • 십부족 : 6장으로 10 이하를 맞추는 경우. 가장 좋은 패.  eg. 1,1,1,1,2,2 또는 1,1,1,1,2,3 또는 1,1,1,1,3,3 또는 1,1,1,2,2,2 또는 1,1,1,2,2,3
  • 스탑 : 6장으로 스트레이트 또는 십부족, 2장으로 5점 이하면 자기 순서가 왔을 때 스탑으로 판을 끌낼 수 있음. 예) A선수가 3,4,5,6,7을 들고 있다가 한장을 가져갔는데 8 또는 2가 나왔으면 그 순간 스탑. A선수가 뻥을 하고 난 후 1,2 두장을 들고 있다가 자기 순서에 스탑.
  • 스탑 바가지  : 두장 들고 스탑을 했는데 더 낮은 점수를 든 사람이 있을 경우 스탑 바가지라고 함. eg. A선수가 1,2 두장을 들고 있다가 스탑을 했는데 화투를 까보니 B선수가 1,1 또는 1,2 들고 있었으면 A선수는 스탑 바가지를 씀. B가 7,7,7,1,2를 들고 있어도 7,7,7은 자연뻥이므로 같은 결과. 자가 버려짐.)

점수를 낮추기 위한 방법

  • 다른 사람에게 뻥을 씌워 두장의 화투를 버리는 방법.
  • 같은 것 3장을 모아 자연뻥 처리. 같은 모양이 세장 이상이면 계산은 0 로.
  • 6장의 카드의 숫자를 나란히 맞춰서 스트레이트로 마이너스 점수를 얻는 방법.

게임 끝내는 방법

  • 누군가 스탑으로 판을 끝내거나 누구든 바가지를 씌워 손에 든 화투장을 털게 되는 경우 게임이 끝남.
  • 화투장을 한 장 가져가고 난 후 뭔가 맞춰졌을 때 스탑 가능. 스트레이트 또는 또이또이 
  • 여섯장 합이 10점 이하면 스탑 가능. 100점 마이너스.
  • 손에 든 화투장의 합이 일정 점수 (보통은 5이나 인원에 따라 달라짐) 이하면 자기 순서에 스톱가능. (그러나 더 낮은 점수를 든 사람이 있으면 스탑바가지.)

점수계산

  • 바가지 : 30
  • 자연뻥 : 0
  • 스탑바가지 : 50
  • 스트레이트 : 점수 합 마이너스
  • 둘둘둘 또이또이 : 0 
  • 셋셋 또이또이 : 0
  • 넷둘 또이또이 : 50 마이너스
  • 십부족 : 100 마이너스

기타

  • 게임의 재미를 위해 각종 점수의 벌점을 달리 정할 수 있음. eg. 5판 이후는 바가지 따블. 막판 바가지 따블 등.
  • 스탑은 본인 이외에도 뻥 한 사람이 한 명 있어야 한다는 룰 도 있지만 5장 들고 자연뻥+바가지가 있을 수도 있으니 상관 없음.
  • 스탑할 수 있는 점수 - 2명이면  3점 3명3점 4명4점 5.6명이면 5점. 등으로 참여 인원에 따라 다르게 정할 수 있음.
  • 게임 중간에 쌓아둔 화투를 다 사용한 경우, 바닥에 나온 화투를 모아서 다시 돌려 게임을 이어갈 수도 있고, 게임의 스릴을 위해 판을 끝낼 수 도 있음.
