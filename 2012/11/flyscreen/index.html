Security door & Flyscreen


현관의 Security Door와 Backyard로 난 Sliding door에 Flyscreen을 설치하기 위한 작업에 들어갔다.

견적 받기. 구글맵에서 Screendoor Flyscreen등으로 검색. local 업체 위주로 컨택했다. 3주동안 5군데 견적 받고 가장 저렴하고 믿음이 가는 업체로 선택. 제품이 다양해서 견적을 비교하는 것이 의미가 없을 수도 있지만 나는 시큐리티 보다는 파리를 막는게 목적이었으므로 가장 기본 제품으로 견적 받았다.

  1. Hire a Hubby 친절한 핸디맨 아저씨가 오셔서 측정. 앞 뒷문 다 달라고 했는데 앞문 Quote 만 주심. $473 inc GST 좀 어설퍼 보여서 탈락.

  2. Keilor Windows 가족 비즈니스. 그자리에서 Quote 종이에 써줌. $990 inc GST Measuring은 맘에 들었으나 dodgy해서 탈락.

  3. Into Blinds 라디오광고 듣고 전화. 방문해서 측정은 대충하더니 이것저것 샘플 보여주고 정신사납게 함. 기본은$1450, 좋은건 $1650. 말이 많아서 탈락.

  4. Doors Plus 집근처 매장 방문. 최상급 롤링블라인드로 추천해 줌. 약 $2000 예상하라고. 비싸서 탈락.

  5. Bulleen Screens 아자씩 방문해서 능숙하고 꼼꼼하게 사이즈만 딱 재고 가심. 전화로 Quote $896. 제일 싸기도 했지만 Measuring 하는 거 보니 신뢰가 가서 선택. 10월 26일 Quote 받고 바로 installation 신청. 오늘 11월 7일 설치하러 옴.

그리고 설치. 실제 작업 시간은 30분정도 걸렸다.
